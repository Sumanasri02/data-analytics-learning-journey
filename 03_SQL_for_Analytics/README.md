# SQL for Data Analytics

## Overview
SQL (Structured Query Language) is a core skill for
data analysts, data scientists, and machine learning engineers.

Most real-world data is stored in databases, and SQL is used to
retrieve, filter, and summarize this data efficiently.

---

## Why SQL is Important for Analytics

- Data is usually stored in databases
- Faster than loading everything into Python
- Used in almost every company
- Essential for dashboards and reports

---

## Core SQL Concepts

### Basic Queries
- SELECT
- WHERE
- ORDER BY
- LIMIT

---

### Filtering & Conditions
- AND, OR, NOT
- IN
- BETWEEN
- LIKE

---

### Aggregations
- COUNT
- SUM
- AVG
- MIN, MAX
- GROUP BY
- HAVING

---

### Joins
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN

Used to combine data from multiple tables.

---

### Subqueries
- Nested queries
- Used for complex filtering and calculations

---

### Window Functions (Basics)
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- OVER()

Used for analytics without collapsing rows.

---

## SQL in the Analytics Workflow

SQL is commonly used to:
- Extract data from databases
- Create analytical datasets
- Perform quick aggregations
- Support dashboards and reports

---

## How SQL Helps an ML Developer

- Efficient data extraction
- Feature creation at database level
- Reduces data transfer and processing cost
- Better understanding of production data

---

## Key Takeaways

- SQL is a must-have analytics skill
- Strong SQL reduces dependency on Python
- SQL + Python is a powerful combination
