## What is Data Analytics?

Data Analytics is the process of:
- Collecting data
- Cleaning and processing data
- Analyzing patterns and trends
- Generating insights for decision making

Data Analytics answers questions like:
- What happened?
- Why did it happen?
- What can we improve?

---

## Why Data Analytics for an ML Developer?

As an ML developer, understanding data analytics helps to:
- Choose meaningful features
- Detect data quality issues early
- Perform strong Exploratory Data Analysis (EDA)
- Build reliable and interpretable ML models

**Good ML starts with good data understanding.**
