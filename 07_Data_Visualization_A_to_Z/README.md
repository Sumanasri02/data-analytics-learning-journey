# Data Visualization A to Z

## What is Data Visualization?
Data Visualization is the graphical representation of data
to identify patterns, trends, correlations, and outliers.

It transforms raw data into meaningful insights.

---

## Why Data Visualization is Important

- Simplifies complex data
- Helps decision making
- Detects patterns & anomalies
- Improves storytelling
- Essential for EDA and reporting

---

## Common Visualization Libraries

- Matplotlib
- Seaborn
- Plotly
- Pandas built-in plotting

---

## Types of Data Visualizations

### 1. Univariate Analysis
- Histogram
- Box plot
- Bar chart

### 2. Bivariate Analysis
- Scatter plot
- Line plot
- Grouped bar chart

### 3. Multivariate Analysis
- Pair plot
- Heatmap
- Facet grids

---

## Choosing the Right Plot

| Data Type | Best Plot |
|---------|----------|
| Categorical | Bar chart |
| Numerical | Histogram / Boxplot |
| Time Series | Line plot |
| Correlation | Heatmap |

---

## Visualization Best Practices

- Use clear labels
- Avoid clutter
- Choose appropriate colors
- Highlight insights
- Keep plots simple

---

## Tools Used

- Matplotlib
- Seaborn
- Pandas
- Plotly (optional)

---

## Key Takeaways

- Visualization comes before modeling
- Good visuals = better insights
- Always match plot to data type
