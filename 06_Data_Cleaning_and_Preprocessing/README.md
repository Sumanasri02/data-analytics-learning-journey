# Data Cleaning and Preprocessing

## What is Data Cleaning?
Data Cleaning is the process of detecting and correcting
inaccurate, incomplete, or inconsistent data.

Clean data leads to:
- Better insights
- Higher ML model accuracy
- Reliable predictions

---

## Common Data Issues

- Missing values
- Duplicate records
- Outliers
- Incorrect data types
- Inconsistent categories
- Noisy data

---

## Data Cleaning Steps

### 1. Handling Missing Values
- Remove rows/columns
- Mean / Median / Mode imputation
- Forward & backward fill

### 2. Removing Duplicates
- Identify duplicate rows
- Drop duplicates safely

### 3. Fixing Data Types
- Convert strings to numeric
- Convert dates to datetime

### 4. Handling Outliers
- IQR method
- Z-score method
- Capping and flooring

### 5. Encoding Categorical Data
- Label Encoding
- One-Hot Encoding

### 6. Feature Scaling
- Normalization
- Standardization

---

## Difference Between Cleaning & Preprocessing

| Data Cleaning | Preprocessing |
|--------------|---------------|
| Fix errors | Prepare data for ML |
| Remove noise | Scale & encode |
| Handle missing values | Transform features |

---

## Importance for ML Models

- Reduces bias
- Improves convergence
- Prevents model failure
- Increases accuracy

---

## Tools Used

- Pandas
- NumPy
- Scikit-learn

---

## Key Takeaways

- Clean data > complex models
- Preprocessing depends on ML algorithm
- Always clean before modeling
