# Excel for Data Analytics

## Overview
Microsoft Excel is one of the most widely used tools for
data analysis in companies across all industries.

Even as an ML developer, Excel is important for:
- Quick data exploration
- Data cleaning
- Business reporting
- Understanding raw datasets before automation

---

## Why Excel is Important for Data Analytics

- Easy to use and widely available
- Fast for small and medium datasets
- Used by business teams and management
- Often the first place where data analysis starts

---

## Core Excel Concepts for Analytics

### Basic Operations
- Sorting and filtering
- Conditional formatting
- Data validation

---

### Essential Formulas
- SUM, AVERAGE, COUNT
- IF, AND, OR
- VLOOKUP / XLOOKUP
- INDEX & MATCH

---

### Data Cleaning in Excel
- Removing duplicates
- Handling missing values
- Text functions (LEFT, RIGHT, TRIM)
- Converting data types

---

### Pivot Tables
- Summarizing large datasets
- Grouping data
- Calculating KPIs
- Creating quick reports

---

## Excel in the Analytics Workflow

Excel is often used to:
- Inspect raw data
- Perform initial cleaning
- Share insights with non-technical stakeholders
- Validate results before using Python or SQL

---

## How This Helps an ML Developer

- Better understanding of datasets
- Faster debugging of data issues
- Improved communication with business teams
- Strong foundation before advanced tools

---

## Key Takeaways

- Excel remains a critical analytics tool
- Strong Excel skills improve data intuition
- Excel complements Python and SQL
